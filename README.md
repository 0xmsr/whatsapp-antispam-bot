# WA-antispam-bot

Script browser console untuk membalas otomatis pesan spam/penipuan di WhatsApp Web.

## Cara Pakai
1. Buka https://web.whatsapp.com
2. Tekan F12 → tab Console
3. Paste isi `script.js` → Enter
4. Buka chat dari nomor spam

## Menghentikan Bot
observer.disconnect();

## Disclaimer
Script ini hanya untuk tujuan edukasi dan perlindungan pribadi dari spam.
Gunakan secara bertanggung jawab.
